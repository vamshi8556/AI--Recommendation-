# AI Recommendation System Using Java

## 📌 Project Overview
This project is a simple AI Recommendation System developed in Java. It recommends suitable technologies or career paths based on user interests and preferences. The project demonstrates the basic concepts of recommendation systems using logical pattern matching. It was developed as part of the DecodeLabs Artificial Intelligence Internship.

## 🎯 Objective
- Accept user preferences or interests as input
- Match the input with predefined data
- Generate relevant recommendations
- Demonstrate the fundamentals of AI recommendation systems

## ✨ Features
- Accepts user input
- Matches preferences using logical conditions
- Recommends relevant technologies or career options
- Simple and beginner-friendly implementation
- Console-based application

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework
- Scanner Class
- Basic Recommendation Logic

## 📂 Project Structure
```
src/
 ├── RecommendationSystem.java
```

## ▶️ How to Run
1. Clone this repository.
2. Open the project in IntelliJ IDEA or Eclipse.
3. Compile and run `RecommendationSystem.java`.
4. Enter your interest when prompted.
5. View the recommended technologies or career options.

Recommended Technologies:
